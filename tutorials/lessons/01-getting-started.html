Lesson 1: Getting Started with Python

In this lesson: We'll learn what Python is, how to install it, and how to write your very first Python program.

What is Python?

Python is a popular programming language that's easy to learn and used by many people around the world. It's like a special language you can use to talk to your computer and tell it what to do!

Python is used for many things like:

  • Creating websites
  • Analyzing data
  • Making games
  • Controlling robots
  • And much more!

People love Python because it's:

  • Easy to read: Python code looks a lot like English, which makes it easier to understand.
  • Beginner-friendly: You don't need to write a lot of code to get things done.
  • Powerful: Despite being easy to learn, Python can do amazing things!

Installing Python

Before you can start writing Python code, you need to install Python on your computer.

Good news: For this tutorial, you don't need to install Python! You can use the interactive code editors on this page to write and run Python code directly in your web browser.

However, if you want to install Python on your computer for more serious projects, follow these steps:

  1. Go to the official Python website: python.org/downloads
  2. Download the latest version for your operating system (Windows, Mac, or Linux)
  3. Run the installer and make sure to check the box that says "Add Python to PATH" during installation
  4. Follow the installation instructions

Important: On Windows, don't forget to check "Add Python to PATH" during installation! This allows you to run Python from anywhere on your computer.

Your First Python Program

Now let's write your very first program in Python! Traditionally, the first program in any language is a simple "Hello, World!" program that displays that message.

Here's how to write it in Python:

print("Hello, World!")

That's it! Just one line of code. Let's break down what this does:

  • print is a Python command (or "function") that tells the computer to display something.
  • ("Hello, World!") is what we want to display. The text is enclosed in quotation marks.
