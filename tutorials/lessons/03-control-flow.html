Lesson 3: Control Flow

In this lesson: We'll learn how to make your programs make decisions and respond differently based on conditions.

Making Decisions with Conditional Statements

In real life, we make decisions all the time based on conditions:

  • IF it's raining, THEN I'll bring an umbrella.
  • IF it's cold, THEN I'll wear a jacket, ELSE I'll wear a t-shirt.
  • IF I'm hungry AND I have food, THEN I'll eat.

Python allows your programs to make decisions in a similar way using if statements.

Basic If Statements

The simplest form of decision-making in Python uses the if statement:

if condition:
    # Code to run if the condition is True

For example:

# Simple if statement
temperature = 30  # temperature in Celsius

if temperature > 25:
    print("It's hot today!")
    print("Remember to drink water.")

Important: In Python, indentation (the spaces at the beginning of a line) is crucial! The indented lines after the if statement only run if the condition is True.

If-Else Statements

Often you want your program to do one thing if a condition is True, and something else if it's False. This is where else comes in:

# If-Else statement
temperature = 20  # temperature in Celsius

if temperature > 25:
    print("It's hot today!")
    print("Remember to drink water.")
else:
    print("It's not very hot today.")
    print("Enjoy the mild weather!")

If-Elif-Else Statements

What if you have more than two possibilities? That's where elif (short for "else if") comes in:

# If-Elif-Else statement
temperature = 20  # temperature in Celsius

if temperature > 30:
    print("It's very hot today!")
    print("Stay hydrated and avoid the midday sun.")
elif temperature > 20:  # This is checked only if the first condition is False
    print("It's a warm day.")
    print("Perfect for outdoor activities.")
elif temperature > 10:  # This is checked only if the previous conditions are False
    print("It's a bit cool.")
    print("You might need a light jacket.")
else:  # This runs if all conditions above are False
    print("It's cold!")
    print("Wear a warm coat.")

You can have as many elif blocks as you need.

Comparison Operators

To create conditions, we need comparison operators:

Operator Description Example
== Equal to a == b
!= Not equal to a != b
> Greater than a > b
< Less than a < b
>= Greater than or equal to a >= b
<= Less than or equal to a <= b

Common Mistake: Don't confuse == (comparison for equality) with = (assignment). The first checks if two values are equal, while the second assigns a value to a variable.

Logical Operators

Sometimes we need to combine multiple conditions. Python provides logical operators for this:

  • and - True if both conditions are True
  • or - True if at least one condition is True
  • not - Inverts the condition (True becomes False, False becomes True)
# Using logical operators
age = 25
has_license = True

if age >= 18 and has_license:
    print("You can drive a car.")
else:
    print("You cannot drive a car.")

Nested If Statements

You can place if statements inside other if statements to create more complex decision structures:

# Nested if statements
age = 25
has_license = True

if age >= 18:
    print("You are an adult.")
    
    if has_license:
        print("You can drive a car.")
    else:
        print("You cannot drive a car because you don't have a license.")
else:
    print("You cannot drive a car because you are not an adult.")

Practice Exercises

  1. Age Group Categorizer: Write a program that categorizes people into age groups: Child (0-12), Teenager (13-19), Adult (20-64), and Senior (65+).
  2. Movie Ticket Pricer: Create a program that calculates movie ticket prices based on age: Children (0-12): $5, Teenagers (13-17): $8, Adults (18+): $12, and Seniors (65+): $7.
  3. Weather Outfit Recommender: Write a program that recommends what to wear based on temperature and weather conditions (sunny, rainy, snowy).
  4. Password Checker: Create a simple program that checks if a password meets certain criteria: at least 8 characters and contains both letters and numbers.
  5. Grade Calculator: Write a program that converts a numerical score (0-100) to a letter grade (A, B, C, D, F).
