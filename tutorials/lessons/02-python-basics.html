Lesson 2: Python Basics

In this lesson: We'll learn about the basic building blocks of Python programming, like how to store information and perform simple operations with it.

Comments in Python: Adding Notes to Your Code

When you write programs, it's helpful to add notes to explain what your code does. These notes are called comments, and the computer ignores them when running your code.

Think of comments like sticky notes you attach to your code. They help you and others understand what's going on.

# This is a single-line comment in Python
print("Hello, World!")  # You can also put comments at the end of a line

"""
This is a multi-line comment (also called a docstring).
You can write as many lines as you want between the triple quotes.
It's great for longer explanations.
"""

Try it yourself! Use the editor below to practice writing and running comments:

Why use comments? Comments help you remember why you wrote code a certain way. They're like notes to your future self and to others who might read your code.

Variables: Storing Information

Variables are like labeled containers that store data. Imagine having different boxes where you store different items - that's what variables do for your data!

In Python, you create a variable by giving it a name, then using the equals sign (=) to assign a value to it.

# Creating variables - notice the pattern: name = value
name = "Alice"    # A string (text)
age = 25          # An integer (whole number)
height = 5.7      # A float (decimal number)
is_student = True  # A boolean (True or False)

Once you've created a variable, you can use it in your program:

# Using variables
print(name)                   # Prints: Alice
print(age)                    # Prints: 25
print("Name:", name, "Age:", age)  # Prints: Name: Alice Age: 25

# You can change a variable's value anytime
age = 26
print("New age:", age)        # Prints: New age: 26

Think of it like: If you have a box labeled "age", you can put the number 25 inside it. Later, you can replace it with 26.

Naming Variables: Rules to Follow

Variable names in Python need to follow certain rules:

  • Can contain letters, numbers, and underscores (_)
  • Must start with a letter or underscore (not a number)
  • Cannot use Python's reserved words (like print, if, or for)
  • Are case-sensitive (so name and Name are different variables)
# Good variable names
user_name = "John"
age1 = 30
_private_data = "secret"

# Not allowed (would cause errors)
# 1st_place = "Gold"  # Cannot start with a number
# print = "Hello"     # 'print' is a reserved word
# user-name = "John"  # Cannot use hyphens

Tip: Use descriptive variable names that explain what the data represents. user_age is better than just a because it's clear what it means!

Basic Data Types in Python

Python has several basic data types that store different kinds of information:

Data Type Description Examples
Strings Text enclosed in quotes (single or double) "Hello", 'Python', "123"
Integers Whole numbers without decimals 42, -7, 1000
Floats Numbers with decimals 3.14, -0.5, 2.0
Booleans True or False values True, False

Checking Data Types

You can check what type of data a variable holds using the type() function:

# Checking data types
name = "Alice"
age = 25
height = 5.7
is_student = True

print(type(name))       # Prints:  (string)
print(type(age))        # Prints:  (integer)
print(type(height))     # Prints:  (floating-point number)
print(type(is_student)) # Prints:  (boolean)

Working with Strings: Text in Python

Strings are sequences of characters (letters, numbers, symbols) enclosed in quotes. They're how Python represents text.

Creating Strings

# You can use single quotes or double quotes
first_name = "John"
last_name = 'Doe'  # Both work the same way

# For quotes inside strings, use the other type of quote
message1 = "He said, 'Python is fun!'"
message2 = 'She replied, "I agree!"'

# Or escape quotes with backslash
message3 = "He said, \"Python is amazing!\""

Combining Strings (Concatenation)

# Adding strings together (concatenation)
first_name = "John"
last_name = "Doe"

# Using the + operator to combine strings
full_name = first_name + " " + last_name
print(full_name)  # Prints: John Doe

# Another way to combine text and variables (f-strings - Python 3.6+)
greeting = f"Hello, {first_name}! How are you today?"
print(greeting)  # Prints: Hello, John! How are you today?

Tip: The f"..." format (called f-strings) is a convenient way to embed variables inside text!

Useful String Operations

# String methods (functions that work on strings)
name = "John Doe"

# Convert to uppercase or lowercase
print(name.upper())  # Prints: JOHN DOE
print(name.lower())  # Prints: john doe

# Get the length of a string
print(len(name))     # Prints: 8 (includes the space)

# Check if a string contains something
print("John" in name)  # Prints: True
print("jane" in name)  # Prints: False

# Replace parts of a string
print(name.replace("John", "Jane"))  # Prints: Jane Doe

# Split a string into a list
print(name.split())  # Prints: ['John', 'Doe']

Accessing Individual Characters

You can access individual characters in a string using their position (index). In Python, indices start at 0, not 1!

name = "Python"

# Accessing characters by index (position)
print(name[0])  # First character: P
print(name[1])  # Second character: y
print(name[5])  # Last character: n

# Negative indices count from the end
print(name[-1])  # Last character: n
print(name[-2])  # Second-to-last character: o

# Slicing (getting a range of characters)
print(name[0:2])   # Characters from index 0 to 1: Py
print(name[2:5])   # Characters from index 2 to 4: tho
print(name[:3])    # First 3 characters: Pyt
print(name[3:])    # Everything from index 3 onwards: hon

String Indexing:

P y t h o n
0 1 2 3 4 5
-6 -5 -4 -3 -2 -1

Basic Math Operations: Python as a Calculator

Python can perform all the basic math operations you'd expect:

# Basic arithmetic
print(5 + 3)   # Addition: 8
print(10 - 4)  # Subtraction: 6
print(3 * 4)   # Multiplication: 12
print(10 / 3)  # Division: 3.3333333333333335 (always returns a float)

# More operations
print(10 // 3) # Integer division (drops decimal part): 3
print(10 % 3)  # Modulus (remainder after division): 1
print(2 ** 3)  # Exponentiation (2 raised to power 3): 8

# Order of operations is just like regular math (PEMDAS)
print(2 + 3 * 4)    # Multiplication happens first: 14
print((2 + 3) * 4)  # Parentheses change the order: 20

Remember: The order of operations in Python follows PEMDAS:

  • P - Parentheses ()
  • E - Exponents **
  • M/D - Multiplication * and Division / (from left to right)
  • A/S - Addition + and Subtraction - (from left to right)

Using Variables with Math

# Math with variables
x = 10
y = 5

print(x + y)    # 15
print(x - y)    # 5
print(x * y)    # 50
print(x / y)    # 2.0

# Update variables with calculations
score = 0
print("Initial score:", score)  # 0

# Add 10 points
score = score + 10  
print("After adding 10:", score)  # 10

# Shorthand way to add to a variable
score += 5  # Same as: score = score + 5
print("After adding 5 more:", score)  # 15

# Other shorthand operators
score *= 2  # Same as: score = score * 2
print("After doubling:", score)  # 30

Getting User Input: Making Interactive Programs

So far, we've been creating programs with fixed values. But most programs need to get information from the user. Python uses the input() function for this.

# Basic input
name = input("What's your name? ")
print("Hello,", name + "!")

# The input function shows a prompt and waits for user to type something
# Whatever the user types is returned as a string

Important: The input() function always returns a string, even if the user types a number!

Converting Input to Numbers

# Getting numeric input
age_string = input("How old are you? ")
print("You entered:", age_string)

# Try adding 1 to age_string
# This would cause an error: print(age_string + 1)
# Because age_string is a string, not a number

# Convert string to integer
age_number = int(age_string)  # Convert to integer
next_year_age = age_number + 1
print("Next year, you will be", next_year_age, "years old.")

# All in one step
height = float(input("How tall are you in meters? "))  # Convert directly to float
print("In centimeters, that would be", height * 100, "cm")

Real-world example: Let's create a simple tip calculator

# Simple tip calculator
print("Welcome to the Tip Calculator!")

# Get the bill amount and convert to float
bill = float(input("What was the total bill? $"))

# Get the tip percentage and convert to float
tip_percent = float(input("What percentage tip would you like to give? "))

# Calculate tip and total
tip_amount = bill * (tip_percent / 100)
total = bill + tip_amount

# Display results
print(f"Tip amount: ${tip_amount:.2f}")  # :.2f means show 2 decimal places
print(f"Total bill: ${total:.2f}")

Type Conversion: Changing Data Types

Sometimes you need to convert data from one type to another. Python provides built-in functions for this:

# Converting between data types
# String to number
age_str = "25"
age_int = int(age_str)    # Convert to integer: 25
price_str = "19.99"
price_float = float(price_str)  # Convert to float: 19.99

# Number to string
count = 42
count_str = str(count)    # Convert to string: "42"
print("You have " + count_str + " messages.")  # Now we can use + with strings

# Float to integer (cuts off decimal part)
height = 5.9
height_int = int(height)  # Result: 5 (decimal part is removed, not rounded)
print(height_int)

# Boolean conversions
print(bool(0))      # False (zero converts to False)
print(bool(1))      # True (non-zero numbers convert to True)
print(bool(""))     # False (empty string converts to False)
print(bool("text")) # True (non-empty strings convert to True)

Common error: If you try to convert a string that doesn't represent a valid number (like converting "hello" to an integer), Python will raise an error.

Understanding Print Options: Formatting Your Output

The print() function is more flexible than it might first appear:

# Basic print
print("Hello, World!")  # Prints: Hello, World!

# Print multiple items (separated by spaces automatically)
name = "Alice"
age = 25
print("Name:", name, "Age:", age)  # Prints: Name: Alice Age: 25

# Concatenation vs. comma-separated values
print("Age: " + str(age))  # Concatenation requires converting numbers to strings
print("Age:", age)         # With comma, print handles different types automatically

# Changing the separator
print("apple", "banana", "cherry")         # Default: apple banana cherry
print("apple", "banana", "cherry", sep=", ")  # Custom: apple, banana, cherry
print("apple", "banana", "cherry", sep="-")   # Custom: apple-banana-cherry

# Changing the ending (default is newline)
print("Hello", end=" ")
print("World!")  # Prints: Hello World!

# Formatting with f-strings (Python 3.6+)
name = "Bob"
age = 30
print(f"{name} is {age} years old")  # Bob is 30 years old

# Formatting numbers in f-strings
pi = 3.14159
print(f"Pi rounded to 2 decimal places: {pi:.2f}")  # Pi rounded to 2 decimal places: 3.14
print(f"Pi as a percentage: {pi:.1%}")             # Pi as a percentage: 314.2%
